This Blog's Source Code

Josh Wang • May 28

That's right. The base source code is open source. You can easily spin up a blog much like this one, inject it with your own style, and make it yours. Check out some of the sweet features that come out-of-the-box.

  1. Built in React for minimal overhead and cruft
  2. Compiles to static file generation and avoids complex hosting
  3. Wrapped by Next.js to keep the door open for server side rendering and more
  4. Leverages Markdown formatting so that your content is transferrable and sustainable
  5. Provides mobile-first experience with layout and design
  6. Provides SEO best practices with metadata tags, opengraph configuration, and more
  7. Provides default syndication with RSS
